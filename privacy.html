Политика конфиденциальности

сервиса Track Start  ·  Дата публикации: 22 июня 2025 г.

1. Общие положения

1.1. Настоящая Политика описывает порядок сбора, хранения, использования и защиты персональных данных пользователей веб-сервиса Track Start, расположенного по адресу https://trackstart.art.

1.2. Оператором персональных данных является ИП Шумилов Александр Вячеславович (ИНН 524300673160, ОГРНИП 319527500060722).

1.3. Используя Сервис, Пользователь соглашается с условиями Политики. Политика разработана в соответствии с Федеральным законом № 152-ФЗ «О персональных данных».

2. Какие данные мы собираем

2.1. При использовании Сервиса могут собираться следующие данные:

  • Электронный адрес — при обращении в поддержку или оформлении подписки.
  • Данные платёжных транзакций — обрабатываются платёжными операторами (ЮКасса); Оператор не хранит данные банковских карт.
  • Технические данные — IP-адрес, браузер, ОС; собираются в обезличенном виде.
  • Данные об использовании — счётчик генераций и лицензионный ключ; хранятся локально в браузере (localStorage) и не передаются Оператору.

2.2. Оператор не собирает специальные категории персональных данных.

3. Цели обработки данных

  • исполнение договора и предоставление доступа к Сервису;
  • обработка платежей и выдача лицензионных ключей;
  • ответы на обращения Пользователей;
  • обеспечение безопасности и предотвращение злоупотреблений;
  • улучшение Сервиса на основе обезличенной статистики.

4. Хранение и защита данных

4.1. Оператор принимает технические и организационные меры для защиты данных от несанкционированного доступа.

4.2. Персональные данные хранятся не дольше, чем необходимо для выполнения указанных целей.

5. Передача данных третьим лицам

5.1. Оператор не продаёт и не передаёт персональные данные в коммерческих целях. Данные могут передаваться:

  • ЮКасса — для обработки платежей.
  • Anthropic, Inc. — запросы передаются через Claude API для генерации текста. Обрабатываются согласно политике Anthropic.
  • Государственные органы — по требованию закона.

6. Файлы cookie

6.1. Сервис использует технические cookie-файлы, необходимые для работы сайта. Они не содержат личной информации.

7.1. В соответствии с ФЗ № 152 Пользователь вправе получить информацию о своих данных, потребовать их уточнения или удаления, отозвать согласие на обработку.

7.2. Обращения направляются на 7002d@mail.ru. Срок рассмотрения — 30 дней.

8. Дети

8.1. Сервис не предназначен для лиц младше 18 лет. Данные несовершеннолетних удаляются незамедлительно при обнаружении.

9. Изменения Политики

9.1. Новая редакция вступает в силу с момента публикации на сайте. Продолжение использования Сервиса означает согласие с изменениями.
